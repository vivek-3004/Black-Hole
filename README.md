## 🧠 About The Project

**THE ABYSS** is a high-fidelity **interactive 3D black hole simulation** that lets users explore the physics of space in a cinematic way.

This project visualizes complex astrophysics concepts like:

* 🌑 Event Horizon
* ⏳ Gravitational Time Dilation
* 🌀 Singularity
* 📏 Schwarzschild Radius

Built using modern web technologies, this project transforms learning into an immersive experience.

---

## 🔍 Features

* 🎮 Interactive 3D environment
* 🌌 Realistic black hole visualization
* ⚡ Smooth animations using Three.js
* 🧪 Physics-inspired simulation
* 📱 Responsive design (works on all devices)
